
# 59.螺旋矩陣II

[力扣题目链接](https://leetcode.com/problems/spiral-matrix-ii/)

给定一个正整数 n，生成一个包含 1 到 n^2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。

示例:

输入: 3
输出:
[
 [ 1, 2, 3 ],
 [ 8, 9, 4 ],
 [ 7, 6, 5 ]
]

## 核心概念
這題參考了leetcode上的官方解答，使用layer概念。也就是像洋蔥一樣，一層一層由外向內層處理。
使用矩陣的方式，從外圈開始，順時針的順序來完成這題。
**程式語言中矩陣[列][行]，與數學中(x,y)相反**
主要需要注意幾個地方:
1. 定錨: 因為題目是要正方形矩陣，所以要先明確定義四個角的座標
- 左上也就是起點(layer, layer)
- 右上(layer, n-1-layer)
- 右下(n-1-layer, n-1-layer)
- 左下(n-1-layer, layer)

2. 轉彎時要跳過已經處理的座標格
- 右上的下面那格(layer+1, n-1-layer)
- 右下的左邊那格(n-1-layer, n-1-layer-1)
- 左下上面那格(n-1-layer-1, layer)
- 左上下面一格(layer+1, layer)

## 程式碼
```
const { json } = require("react-router-dom");

/**
 * @param {number} n
 * @return {number[][]}
 */
var generateMatrix = function (n) {
    // 初始化一個nxn的空矩陣 
  let result = Array.from({ length: n }, () => Array(n).fill(0));
    // 初始化計數器，代表下一個要填入矩陣的數字
  let cnt = 1;
  let target = n*n
    // 層級，代表指針(ptr)目前在哪一層
  let layer = 0;

  // 用layer迴圈遍歷所有層的運算
  for(layer = 0; cnt<=target; layer++){
  // 方向1: 由左至右，指針(ptr)從左上座標(layer,layer)到右上座標(n-1-layer, layer)
  for (let ptr = layer; ptr <= n - 1 - layer; ptr++) {
    result[layer][ptr] = cnt++;
  }
   // 方向2: 由右上至右下，指針(ptr)從右上的下面那格(n-1-layer, layer+1)開始到右下座標(n-1-layer, n-1-layer)
  for(let ptr = layer+1; ptr <= n -1- layer; ptr++){
    result[ptr][n-layer-1] = cnt++;
  }

   // 方向3: 由右下左邊一格(n-1-layer, n-1-layer-1)至左下(n-1-layer, layer)
   for(let ptr = n-1-layer-1; ptr >= layer; ptr--){
      result[n-1-layer][ptr] = cnt++;
   }

   // 方向4: 由左下上面一格(n-1-layer-1, layer)至左上下面一格(layer+1,layer)
   for(let ptr = n-1-layer-1; ptr >= layer+1; ptr--){
      result[ptr][layer] = cnt++;
   }

  }
  
 
  console.log("目前矩陣狀態:");
  console.log(result);
  return result;
};
generateMatrix(5);

```
