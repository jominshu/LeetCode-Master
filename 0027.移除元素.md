## 題目

[力扣題目連結](https://leetcode.com/problems/remove-element/description/)

給一個要刪除的元素值，從一個數組中刪除該元素。
核心概念: 數組不能夠直接刪除，因為在記憶體中數組是連續的，只能用覆蓋的方式。
### 算法
使用雙指針法，給一個快指針跟慢指針。快指針負責遍歷整個數組，慢指針負責更新數組

### 程式碼
```
var removeElement = function(nums, val){
    // k是慢指針的起始下標位置
    let k = 0; 
    // 這是快指針遍歷數組，i代表快指針的下標位置
    for(let i = 0; i < nums.length; i++){
    // if判斷數組元素i是否為要刪除的元素val，若不是要刪除的元素，就要放入到慢指針的位置k進行覆蓋
    if(nums[i] != val){
        nums[k] = nums[i];
        // 覆蓋完後要將慢指針也向前一格
        k++
    }
}
// 迴圈結束後要根據題目return k，也就是刪除完的數組長度
return k;
};
