# 题意：反转一个单链表。

示例:
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

## 思路
使用雙指針法，反轉鍊表依照以下步驟
1. 先用temp將cur.next位置保存 => temp=cur.next
2. cur箭頭反轉，指向pre，此時cur的箭頭從原本指向temp改成指向pre => cur.next=pre
3. pre移動到cur位置 => pre= cur
4. cur移動到temp，反轉完成 => cur=temp

## 程式碼

```
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function(head) {
    // 判斷head是否為null或者head.next是否為null(代表只有一個頭節點，不用反轉)，若為true直接return null
    if(!head || !head.next) return null
    // 定義pre, cur, temp初始值
    let pre = null; // 整個Linked List反轉後，pre為null
    let cur = head;
    let temp = null;
    
    while(cur){
        temp = cur.next
        cur.next = pre
        pre = cur
        cur = temp
    }
    return pre
};

```
###
空間複雜度O(n)
時間複雜度O(1)
