# 160. 鏈結串列相交

[力扣题目链接](https://leetcode.com/problems/intersection-of-two-linked-lists/)

给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有交点，返回 null 。

## 思路
使用雙指標，先確認兩個鏈結串列的長度，較長的先跑，指標移動到相同起點位置，開始向後尋找相交的頭節點。

1. 計算長度： 分別走訪兩個鏈結串列，計算出它們的總長度。
2. 比較長度： 找出哪一條鏈結串列比較長。
3. 計算差距： 較長串列的長度 減去 較短串列的長度，得出「長度差（步數）」。
4. 對齊起跑線： 讓指向「較長串列」的指標，先往前移動這個「長度差」的步數。
5. 攜手同行： 此時兩個指標距離終點的距離一模一樣。讓兩個指標同時一步一步往後走。
6. 尋找交點： 當兩個指標踩在同一個節點（記憶體位址相同）時，那個位置就是我們要找的交會點。（如果並肩走到最後變成 null 都沒相遇，代表無交集）。

## 程式碼

```

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} headA
 * @param {ListNode} headB
 * @return {ListNode}
 */

// 宣告計算串列連結的長度函數
var getListNodeLen = function(head){
    // 初始化計數器
    let len = 0;
    // 宣告指標並放到頭節點位置，後續要從頭開始往尾巴走
    let cur = head;
    // 讓節點一步一步向後走，計數器也要加相同步數
    while(cur){
        cur=cur.next;
        len++
    }
    // 回傳總布數就相當於這個鏈結串列的長度
    return len;
}
var getIntersectionNode = function(headA, headB) {
    // 計算headA, headB長度
    let lenA = getListNodeLen(headA);
    let lenB = getListNodeLen(headB);

    // 宣告curA, curB並放在頭節點headA, headB
    let curA = headA
    let curB = headB
    // 比較lenA, lenB長度，若headB比較長，交換curA跟curB,lenA跟lenB
    if(lenA < lenB){
        [curA, curB] = [curB, curA];
        [lenA, lenB] = [lenB, lenA];
    }

    // 計算較長的lenA要先偷跑幾格(上面已交換，確保lenA是較長的)
    let i = lenA - lenB;
    while(i-->0){
        curA = curA.next;
    }

    // 現在curA, curB在相同起點要一起往後走到結尾,
    // 當curA, curB交會時跳出循環，會是沒交會回傳null
    while(curA && curA!==curB){
        curA = curA.next;
        curB = curB.next;
    }
    return curA;


};

```
