[力扣题目链接](https://leetcode.com/problems/minimum-size-subarray-sum/description/)

给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。

示例：

* 输入：s = 7, nums = [2,3,1,2,4,3]
* 输出：2
* 解释：子数组 [4,3] 是该条件下的长度最小的子数组。

提示：

* 1 <= target <= 10^9
* 1 <= nums.length <= 10^5
* 1 <= nums[i] <= 10^5

程式碼：
```
var minSubArrayLen = function(nums, target){
  let left = 0;
  let sum = 0;
  let minlen = Infinity;
  // 先遍歷整個數組，滿足第一個條件>=target
  for(let right = 0; right < nums.length; right++){
    sum += nums[right];
      // 當快指針找到的sum>target時，要更新慢指針進行內縮
      while(sum >= target){
        // 這裡是比較原本的minlen跟新找出的sublen哪個比較小
        minlen = Math.min(minlen, right - left +1)
        // 找到較小的sublen後要減去最左邊的慢指針的元素，更新sum的值，以便下一輪遍歷找到>=target的新sublen
        sum -= nums[left]
        // 最左邊的慢指針要往前一格，內縮sublen範圍
        left++;
    }
  }
  // 條件 ? 成立回傳值 : 不成立回傳值（三元運算子） 
  return minlen === Infinity ? 0 : minlen;
}
```
